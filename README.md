# Automate the Boring Stuff Exercises

Practice exercises from *Automate the Boring Stuff with Python* by Al Sweigart.

## Overview
This repository contains exercises and projects from the book. The goal is to build practical skills in Python automation and scripting.

## Contents
Each chapter has its own folder with the related exercises:

- **Chapter 1:** [Introduction to Python]
- **Chapter 2:** [Flow Control]
- **Chapter 3:** [Functions]
- ... *(...In progress)*

## Usage
Feel free to review, modify, and run the code for learning purposes. Each file is self-contained and can be run independently.

## Requirements
Python 3.x is recommended to run these scripts. Some exercises may require additional libraries.
